# YFL_Email_Form_Guide
YFL Form Guide Automation — Playwright scraper that logs into YFL LeagueHub, extracts standings, fixtures, and generates an enhanced full-season form guide (W/L/D/N/V) for U11 Divisions 1–3. Builds a rich HTML report and emails it via Gmail API. Runs weekly on GitHub Actions and can be customized for any age group or division.
